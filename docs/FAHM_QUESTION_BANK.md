# Fahm Test — Question Bank (فهم)

> **320 psychometric questions** testing how connected your worldview is to the Quran's absolute truth. No right or wrong — only self-awareness.

**Scoring:** Each answer carries a **Quranic Alignment Score (QAS)** from 1–4.
- **1** = Furthest from Quranic worldview (ego/dunya-driven)
- **2** = Leaning toward dunya with some awareness
- **3** = Transitional — aware but struggling
- **4** = Most aligned with Quranic truth (tawakkul/taqwa-driven)

> **Options are intentionally shuffled.** The "right" answer is not always the last one. The test is about self-reflection, not getting a high score.
